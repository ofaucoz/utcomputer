Detailed Description

RepositoryObserver are the "Observers" elements of the Observer pattern: they are attached to repositories to be notified when they change.

The user interface widgets are implementing this interface to re-render on data change.

This method will be called by the repositories which this observer is attached to. This method is the place where observers should update themselves according to the new data.
