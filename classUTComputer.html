Detailed Description

This class is the main entrypoint of the UTComputer library.

It encapsulates the Lexer, the Resolver and the Runner to fully execute a string command. It can also handle already parsed commands (for instance, literals provided by the Eval operator).

This class does not return any value : it will change the models data and therefore the results will be provided using the Observer design pattern.

void UTComputer::execute ( string  command)
inline

Execute a command given as a string.

This will tokenize the command using the Lexer, resolves the atoms of this command using the Resolver and apply the command to the stack using the Runner.

Exceptions
InvalidSyntaxExceptionWhen the given command has an invalid syntax.
UndefinedAtomExceptionWhen there is an undefined atom used in the command.
UnsupportedLiteralExceptionWhen a literal that can't be handled by the runner is found.
