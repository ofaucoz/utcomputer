Detailed Description

The Runner is the third and last main component of the UTComputer library.

Called after the Resolver, its role is to apply the command to the stack.

It uses the operators list to execute actions on the stack and it pushes other literals on top of it.

Member Function Documentation

void Runner::run ( LiteralVector  resolvedTokens)

This is the main method of the Runner.

Using the operators list and the provided literals, the runner applies the asked operations to the stack.

If an unkown literal that can't be handled is found, it throws an UnsupportedLiteralException.
